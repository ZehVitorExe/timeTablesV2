
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Description

O presente projeto se trata de uma API Rest de gerencia de eventos, como um timeTables, podem ser cadastrados palestrante, eventos e sessões

Rotas de listagem, login e registro de usuario são publicas as demais devem usar token obtido pelo login

Para exemplos consulte curl.md
